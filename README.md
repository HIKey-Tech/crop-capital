# CropCapital Monorepo

This repository is the unified monorepo for CropCapital.

## Structure

- `apps/backend` — Express + TypeScript API
- `apps/frontend` — React + TanStack frontend
- `packages/*` — shared libraries (future)

## Quick start

```bash
pnpm install
pnpm dev
```

## Common commands

```bash
pnpm build
pnpm typecheck
pnpm test
pnpm lint
```
